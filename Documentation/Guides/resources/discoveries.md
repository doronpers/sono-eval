## üíª CLI Discoveries

### `sono-eval insights --deep`

**How to discover:** Run this command in your terminal

**What you unlock:** 
- Raw metric data access
- Cross-path comparison tools
- Trend analysis
- Pattern detection algorithms

**Value:** Expert-level analysis from the command line

### `sono-eval insights --pattern-recognition`

**How to discover:** Run this command

**What you unlock:**
- Design pattern detection
- Anti-pattern identification
- Code smell analysis
- Best practice recommendations

**Value:** Advanced code analysis features

### `sono-eval hidden --secret`

**How to discover:** Run `sono-eval hidden --secret` or `sono-eval --secret`

**What you unlock:** Secret configuration options:
- `EXPERT_MODE=true`
- `DEBUG_TRACKING=true`
- `ADVANCED_ANALYTICS=true`
- `PATTERN_DETECTION=true`

**Value:** Enable advanced features via configuration

### `sono-eval hidden --explore`

**How to discover:** Run this command

**What you unlock:** List of all discoverable features

**Value:** Guide to finding all easter eggs

## üîç How Discoveries Work

1. **Exploration Rewarded:** When you discover a feature, you get a notification
2. **Persistence:** Discoveries are saved in your browser (localStorage)

4. **No Spoilers:** Features aren't advertised - they reward curiosity
